# Virtual Art Museum

A simple web application to explore a curated collection of high-quality paintings from the Art Institute of Chicago.

## Overview

This project provides an online interface to browse artworks. It is designed to fetch and display paintings, allowing users to click on any piece to view it in high resolution. The page is built to be a clean and direct gallery experience.

## Features

*   **Curated Collection**: Displays paintings sourced from the Art Institute of Chicago's collection.
*   **High-Resolution Viewing**: Click on any artwork to see it in high detail.
*   **Simple Interface**: Focuses on the art with a minimalist design for easy browsing.

## How to Use

1.  Open the webpage in your browser.
2.  Browse the displayed artworks.
3.  Click on any artwork you are interested in to view a high-resolution version of it.

## Technical Implementation

*   The application fetches artwork data from the **Art Institute of Chicago's public API**.
*   The page is a static site hosted on **GitHub Pages**, making it fast and reliable.
*   It is built with standard web technologies (HTML, CSS, JavaScript) to ensure broad compatibility.
